# Программный модуль для поддержки и тестирования программных компонентов

Данный проект разработан в рамках производственной практики.

## Описание
Программа позволяет выполнять тестирование программных модулей различных типов, анализировать результаты и сохранять их.

## Возможности
- Создание модуля
- Тестирование
- Проверка корректности данных
- Сохранение результатов в JSON

## Технологии
- Python
- JSON
