# Программный модуль для поддержки и тестирования программных компонентов

Данный проект был разработан в рамках производственной практики.

## Описание
Программа предназначена для тестирования различных программных модулей и анализа их работы.

## Возможности
- Выбор типа модуля
- Запуск тестирования
- Вывод результатов
- Сохранение результатов в файл

## Технологии
- Java
- Работа с файлами
- Основы тестирования ПО
